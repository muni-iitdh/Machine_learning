# Machine Learning from Scratch

Building ML algorithms using only NumPy to understand the mathematics behind them.

## Projects

### 1. Linear Regression
- Implemented gradient descent with MSE loss
- California Housing dataset
- R² Score: 0.58
- Features: model class, loss history tracking, learning rate experiments

### 2. Logistic Regression
- Binary cross-entropy loss with sigmoid activation
- Breast Cancer Wisconsin dataset
- Accuracy: 97.4% | AUC-ROC: 0.996
- Features: probability calibration, confusion matrix, ROC curves

## Key Learnings
- Gradient computation for different loss functions
- Feature scaling importance for gradient descent convergence
- Bias-variance tradeoff analysis
- Evaluation metrics: R², precision, recall, AUC-ROC

## Tech Stack
- Python, NumPy, Pandas, Matplotlib, Scikit-learn (for comparison only)
